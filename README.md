# Android_Development
**This demo project is built on the following topics:**

- UI Widgets
- View Layouts
- Constraints layout
- Navigation component
- RecyclerView and Adapter
- View Pager
- Activity
- Fragment
- Intent
- Web Service
- Retrofit
- OKHttp3
- JSON parsing using GSON
- Coroutines
- Dependency injection with Hilt
- Repository pattern
- MVVM Architecture
